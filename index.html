IOWEU is a free, web-based group expense tracker that helps friends, roommates, and travel groups split bills, track shared costs, and settle debts in real time.

What Is IOWEU?

IOWEU (pronounced "I owe you") is a free group expense tracking platform built for splitting shared costs. The name is a play on the well-known IOU abbreviation, representing its core purpose: tracking who owes whom in any group.

Unlike traditional expense-splitting apps, IOWEU runs entirely in a web browser — no app download is required. Users create a "room," invite group members via a unique room key, log expenses as they happen, and IOWEU automatically calculates every member's balance in real time.

What Does IOWEU Do?

IOWEU simplifies group finances by automating expense tracking and debt calculation. Here is what the platform offers:

Room-Based Groups

Create private rooms for each group — a trip, a flat, a project — and share a unique key so members can join instantly.

Expense Logging

Log every shared cost with a title, amount, date, and who paid. IOWEU records it and updates all balances immediately.

Equal & Custom Splits

Split expenses equally among all members or set custom amounts for each person when costs are not evenly shared.

Real-Time Debt Board

See exactly who owes whom and how much at any moment. Balances update live as new expenses are added.

Amount Builder

A tactile dial interface for building expense amounts from denomination buttons — especially useful on mobile devices.

Multi-Currency Display

Each user sets their display currency at sign-up, so amounts are always shown in a familiar format.

How IOWEU Works

Getting started takes less than a minute:

Sign Up for Free

Create an account with your name, email, and preferred currency. No credit card or phone number required.

Create or Join a Room

Create a new room and set a room key, or join an existing room by entering a key shared by a group member.

Log Shared Expenses

Add expenses as they happen. Enter the title, amount, date, who paid, and choose equal or custom split.

Track & Settle Debts

IOWEU calculates every member's net balance automatically. View the debt board to see who owes whom and settle up.

Frequently Asked Questions About IOWEU

What does IOWEU mean?

IOWEU stands for "I Owe You." It is a stylized version of the IOU abbreviation, reflecting the platform's purpose of tracking debts and shared expenses within groups.

Is IOWEU free?

Yes. IOWEU is completely free to use. There are no premium tiers, no advertisements, and no hidden charges. All features are available to every user.

Do I need to download an app?

No. IOWEU is a web application that runs entirely in your browser. Visit ioweu.co.in from any device to start using it.

How is IOWEU different from Splitwise?

IOWEU is a lightweight, browser-based alternative to apps like Splitwise. It requires no download, has no paywall, and focuses on simplicity — create a room, add expenses, and see balances instantly.

Who made IOWEU?

IOWEU was created by Truculence Studio, an independent software studio that builds practical, well-designed web tools.

Is my data safe on IOWEU?

IOWEU uses Supabase (built on PostgreSQL) for secure data storage with row-level security. User authentication is handled with industry-standard practices.
